### 표

> `>`로 인용문구 만들 수 있는데, shift + enter로 다음줄 가기 가능
>
> ctrl + t로 표 만들 수 있다.
>
> | 순번 | 이름 |      |
> | ---- | ---- | ---- |
> | 1    | 이   |      |
> | 2    | 정   |      |
> | 3    | 원   |      |
>
> 

취소선 `~~`사이에 넣기

이탤릭(기울이기) `**`사이에 넣기
